# Bulls-and-Cows
アメリカの大学の講義課題としてJavaを使って「Bulls and Cows」というゲームを再現しました。（2026年1月頃）

## 概要
数字を当てるゲームの再現をJavaで行いました。

## Bulls and Cowsのルール
1.　0から9までの数字で、すべて異なる4桁（10桁以下で設定）の数字を決めます。
2.　挑戦できる回数（ラウンド数）を設定します。
3.　プレイヤーは数字を推測し、4桁の数字をコールします。
4.　2つのヒント（インジケーター）を数えてプレイヤーに返します。
  　Bull（ブル / 雄牛）: 数字も位置もぴったり合っている数
  　Cow（カウ / 雌牛）: 数字は合っているが、位置が違う数
  　例：秘密の数字が1368の場合
  　プレイヤーの推測:1476
  　→Bulls:1 Cows:1
  　プレイヤーの推測:2369
  　→Bulls:2 Cows:0

5.　設定した挑戦回数内で全ての数字を当てる（4桁の場合は4 Bulls）ことができればプレイヤーの勝利です。
